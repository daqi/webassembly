# 定义
https://webassembly.org/
WebAssembly（缩写为Wasm）是基于堆栈的虚拟机的二进制指令格式。
Wasm被设计为可编译等级语言（如C/C++/Rust）的可移植目标，从而可以在Web上为客户端和服务器应用程序进行部署。
