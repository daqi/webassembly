# 定义

WebAssembly（缩写为Wasm）是基于堆栈的虚拟机的二进制指令格式。
Wasm被设计为可移植目标，支持编译高级语言（如C/C++/Rust），从而可以在Web上为客户端和服务器应用程序进行部署。
